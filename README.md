# CSharpTrainingConsoleApp

Консольное приложение для практики работы с исключениями и событиями на C#. Проект содержит два самостоятельных задания с возможностью выбора через главное меню.

---

## Структура проекта

```
CSharpTrainingConsoleApp/
│
├─ Program.cs                  # Главная точка входа, меню выбора заданий
├─ MyCustomException.cs        # Собственный тип исключения для задания 1
├─ InvalidSortOptionException.cs  # Собственное исключение для задания 2
├─ NameSorter.cs               # Класс сортировщика фамилий со событием
└─ README.md                   # Документация проекта
```

---

## Описание заданий

### Задание 1 — Массив исключений

* Создан массив из 5 действий (`Action[]`), каждое из которых при выполнении вызывает определённое исключение:

  * `ArgumentNullException`
  * `IndexOutOfRangeException`
  * `DivideByZeroException`
  * `InvalidOperationException`
  * `MyCustomException` (собственный тип)
* Все действия выполняются в цикле с конструкцией `try/catch/finally`.
* В `catch` выводится тип исключения и сообщение.
* В `finally` демонстрируется блок для логирования или освобождения ресурсов.

**Цель:** освоить создание собственного типа исключения и обработку разных исключений в цикле.

---

### Задание 2 — Сортировка фамилий через событие

* Список из 5 фамилий:

  ```
  Кузнецов, Иванов, Петров, Сидоров, Алексеев
  ```
* Пользователь вводит:

  * `1` → сортировка А–Я
  * `2` → сортировка Я–А
* Сортировка реализована через **событие** `OnSort` класса `NameSorter`.
* Проверка введённых данных:

  * Если введено не число → `FormatException`
  * Если введено число, но не `1` или `2` → выбрасывается `InvalidSortOptionException` (собственное исключение)
* После сортировки список выводится в консоль.

**Цель:** практика работы с событиями, обработкой пользовательского ввода и собственными исключениями.

## Как запустить

1. Склонируйте или скачайте репозиторий.
2. Откройте проект в Visual Studio / Rider / VS Code.
3. Соберите проект.
4. Запустите приложение.
5. В главном меню выберите задание:

   * `1` — демонстрация исключений
   * `2` — сортировка фамилий
   * `0` — выход из программы

## Особенности реализации

* Массив действий (`Action[]`) для демонстрации разных исключений
* Событие `OnSort` в классе `NameSorter` позволяет динамически подписывать методы сортировки
* Собственные исключения:

  * `MyCustomException` — для задания 1
  * `InvalidSortOptionException` — для задания 2
* Простое и удобное меню для выбора задания
* Используется LINQ для сортировки списка фамилий
